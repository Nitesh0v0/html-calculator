# Simple HTML Calculator
A clean and functional web calculator built with HTML, CSS, and JavaScript.

## How to Run
1. Download the files.
2. Open `https://nitesh0v0.github.io/html-calculator/` in your browser.

## Features
* **Live Math**: Performs addition, subtraction, multiplication, and division.
* **Responsive**: Works on both mobile and desktop screens.
* **Modern Design**: High-contrast buttons for better visibility.
